# Day 7 - Server and its services

- Daemon is a computer program that runs in the background, handling tasks without user interaction
- Apache is a web server
- Installed apache using `sudo apt install apache2`
- This turns your server into a web server
- This will run on your public IPv4 address
- Apache provides a service for http access
- `sudo systemctl status apache2` to check apache server status
- you can use `systemctl` to start or stop apache
